# Project Checking Steps

## Admin Credentials

- Email: `admin@storerating.com`
- Password: `Admin@123`

## Steps to Check the Project

1. Open the frontend URL.
2. Log in with the admin credentials given above.
3. Verify the admin dashboard shows:
   total users, total stores, and total ratings.
4. Go to the users section and verify:
   admin can add users and filter users by name, email, address, and role.
5. Go to the stores section and verify:
   admin can add stores and view store listings.
6. Log out from the admin account.
7. Sign up as a normal user from the signup page.
8. Log in as the normal user.
9. Open the stores page and verify:
   store list is visible, search works, and ratings can be submitted or modified.
10. Update the password from the settings page.
11. Log out from the normal user account.
12. Log in as a store owner account created by admin.
13. Verify the store owner dashboard shows:
    average rating and users who submitted ratings.
14. Check the backend health URL to confirm backend is running.
